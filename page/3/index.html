前端基础之HTML

2016-02-03

HTML XHTML XML 的区别？

发展趋势：

html(超文本标记语言)——xhtml(可扩展性超文本标记语言)——xml（可扩展性标记语言）;

html：

1.对大小写不敏感
2.标签不必成对出现 如：<br>

xhtml：

1.对大小写敏感，必须是小写的；
2.标签必须成对出现，有开始标签就必须有结束标签；
3.属性值必须在引号之内； s
4.不支持属性最小化：
eg：
正确：<input checked='checked'>
错误：<input checked>
5.name属性不赞成使用，以后会被淘汰；
6.空元素也要结束标签：如：<br/>,<hr/>(水平分割线)

xml：

html最终会发展到xml，xhtml是html想xml发展的一个过渡，xhtml的特性也适合xml；

常用的 DOCTYPE 声明

在 HTML 4.01 中，<!DOCTYPE> 声明引用 DTD，因为 HTML 4.01 基于 SGML。DTD 规定了标记语言的规则，这样浏览器才能正确地呈现内容。
HTML5 不基于 SGML，所以不需要引用 DTD。

HTML 5

<!DOCTYPE html>

HTML 4.01 Strict

该 DTD 包含所有 HTML 元素和属性，但不包括展示性的和弃用的元素（比如 font）。不允许框架集（Framesets）。
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

HTML 4.01 Transitional

该 DTD 包含所有 HTML 元素和属性，包括展示性的和弃用的元素（比如 font）。不允许框架集（Framesets）。
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

写给你们的我的梦

2016-01-01

晚上给家里打电话，话没说完，挂了。这是第一次，我妈挂我的电话。

下了象棋至深夜两点，三局两胜，输了。而这已经是第二次输给舍友了。

可能，夜深了，才显得今夜一如既往。也许，冷静了，才会对事物格外的敏感。

夜的一点前，企鹅群以及宿舍都很热闹。微信震动着群发的嘘寒问暖的假意的问候，朋友圈展示的始终是那么几个人发的无关痛痒让早已让人审美疲劳的各种秀。

晚上七点，在我吃惊的发现两个女生在超市买了几瓶啤酒后，便不落俗套的也买了。很可笑，我笑她们或许以为我和她们一样都是想要进行所谓的跨年。很可笑，我笑每年《千里》主持人都要说的那句“时间太瘦、指缝太宽”。很可笑，我笑科技的进步让这个本来就缺乏温情的时代让人们缺乏真正的幸福感。更可笑，我笑自己得不到同情的矫情的感慨，笑自己在你们的眼中居然是和你们一类的人。

时光倒退十五年，我九岁，在老家，五间大瓦屋，时间定格在那年的春到冬。我家门前有口井，井的旁边有两块石磨（早年间用来制造面粉、豆腐一类的东西）和一个“滚子”（碾压稻谷用的），至于它们的年代，反正都要比我老。那个时候，由于那些古玩意儿几乎用不着，妈妈喜欢在那些石头上用“槌棒”洗衣服，邻居家由于没有井，也来我家门口洗。每年冬天，哪天天气好了，就会是我门口最热闹的时候。隔壁的两家邻居，以及我妈，都会用井里的温水来洗衣服，边洗边说笑着。这样场景很美，很温情，我很怀念。

农村的冬天，很闲，最常见的是一个村子里的几个人凑一块打麻将。然后各家的孩子，也就跟着大人也聚到了一起，孩子们在大人打完麻将后，总要问问谁赢谁输，而我，基本每次都能在伙伴们中昂起头。我很崇拜我爸，至少在16岁前一直认为我爸无所不能，认为我爸很神。当时最让我振奋的就是他能在摩托车是个相对比较稀罕的年代，通过打麻将赢钱攒钱，买个摩托车。我爸在村里属于知识分子，是为村里数不多的高中生。他学过修电视，修收音机，我家床的下面有个大木箱，箱子里面全部都是他的工具和一些喇叭，磁铁，收音机电视机的电路板等。小的时候，家里买不起玩具，但是大木箱却是我的宝贝。每次只要有小伙伴来家里，我都会很神秘的从大木箱里拿出点东西来玩。开始是磁铁，后来是小灯泡，再后来是发动机，不亦乐乎。

喂，内个大三的！

2015-09-12

那么多年自作聪明付出了真心
总以为换到一个公平的回应
你床边的蜷曲头发残酷地说明
缠绵的爱比不上一时的高兴
你的多情出卖我的爱情
赔了我的命
我卖了一个世界却换来灰烬

坐在电脑桌前，看着旁边还剩半瓶的农夫山泉，顺便点了下手机，时间是2015年9月2日。是凌晨一点。发呆。

就在刚刚写完了研究会学访的旅途感受记录。此刻窗外能清晰地听见不知道是蛐蛐还是蝈蝈，抑或是其他虫子动物传来的杂七杂八的声音。看着书架上堆着的那些初高中曾经看过的可以说是特别“乱”的书，随手取下一本，掸去灰尘，翻开看着，闻到了那久违的带着湿气却又香香的味道。突然感觉自己好像离那时候的自己已经走得很远了，那时看似可笑的梦想，在现在的我看来是在给自己的左右脸轮回交替来了好多巴掌。

高中励志书看多了，什么“每个人心中都有个巨人，当巨人苏醒时，奇迹就会发生”、“你无法取悦每一个人，当你试着去取悦所有人的时候，你将失去自己”。所以，有时候，看似老实乖巧的我，骨子里却是着狂傲不羁的，很多时候有彪悍的想法，却没有彪悍的实力。评价下自己吧：天生条件很一般，长得比辉辉、鹏举、佳哥此类自称不屌丝实际上就长得屌丝点的基友好点；但是跟香帅、大师这一类人比，我是自己甘愿拜倒在他们的下半身下的。很一般。看上去完全不具有领导力，完全不符合在学校在社会能混的很好的那一种人。所以，进入大学的第一件事就是混，然后参加各种社团，尽量的让自己成为别人眼中的牛人。可以为部长交给的一个策划，屁颠屁颠的去连续熬夜做，期许并盼望着自己也可以当部长，盼望着能做好多让自己想法得到实施的事。虽然后来没有成为部长，但是部员熬夜写策划的日子，每当想起来，都会在心中欣喜一番。初入大学的自己和我现在的男朋友友女朋友们初入大学的样子也算是如出一辙吧。至少会让自己知道如何去把我一场活动的流程安排，至少现在的我依然可能是迂腐的认为，这些经历，对我以后入职公司，替公司出谋划策有一定的帮助。

很多人对我的感情史很感兴趣，至少我自傲的认为有很多人关注过我。对于我的同学朋友们来说，这个话题也许虽然敏感但是不复杂，年轻时候谈过几场恋爱，追过几个女生并不算丢人。对我来说，感觉初入大三的我，如果文学功底在好那么一点的话，已经完全可以根据经历来写书了。我曾深深沦陷过，也曾体无完肤、焦头难额，虽然那个谁特么的居然用我教给她的话对我说：你是个好人。虽然说的是实话，但是当她把这话说出来的时候，气的我把二十年来学的脏话全部在心中翻腾了一遍，翻腾了两个多小时后，给她微信发了个句号，其实我是想发“滚”的，但是她说我是好人，我也该保持风度。其实，很多事，只要不是自己的原因，当你活的坦荡彪悍一点后，你会发现一切都没有什么大不了,人生总有几次踩到大便的时候是不是。也许你会觉得我小子不地道，拐着弯子骂人，但是当你再多经历那么一点点的时候，你会发现我说的并不算过分。后来才真正懂什么叫:做正确的事，比你正确的做事要重要的多；对人也是这样，爱一个人对的人，比单纯的爱一个人要重要的多。面对大学，看似很自由，却也很禁锢。有时觉得傲娇坦荡，狗屁的规矩，疯就疯了，守什么规矩，狂放孤傲一点也会活得很自在。而然，是否目前大学，是否让我忘了我曾经的以为。
